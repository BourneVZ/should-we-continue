# should-we-continue · MMTI

> **Matrescence Mode Type Indicator — 母职转变模式指标**

一个围绕怀孕与未来养育想象场景设计的、基于多维模式匹配的网页类型测试。它不是"恋爱人格换皮"，也不是决策报告工具——它回答的是：**"此刻的你，更容易进入哪一种反应模式。"**

---

## 产品理念

妊娠矛盾心理（pregnancy ambivalence）不是单一的"想要"或"不想要"，而是愿望、失控感、他人影响、自我保护与预期情绪的混合体。母职转变（matrescence）是身体、认知、身份、关系与现实负荷同时变化的复合过程。

因此本项目的底层建模方式为：

```
连续维度 → 反应簇 → 原型类型
```

- **15 个连续维度** 描述用户在五个模型上的倾向分布
- **反应簇（家族）** 将相近的矛盾模式归为一组
- **24+1 个原型类型** 作为前台表达，让结果有记忆点、可传播

**不是心理诊断，不是医学量表，不是决策建议。** 文案路线是"轻毒舌但不冒犯"——有梗、有刺、有自嘲，但不病理化，不对创伤、暴力或精神疾病开玩笑。

## 产品结构

### 测试流程

| 环节 | 内容 |
|------|------|
| **首页** | 品牌介绍、5 大模型说明、4 步原理、25 型预览、开始/继续/重测入口 |
| **答题** | 30 题（18 场景题 + 12 陈述句），5 分量表，约 3–5 分钟完成 |
| **结果** | 匹配类型海报、匹配百分比、15 维雷达图、维度指纹、6 段深度解读 |

### 维度体系

| 模型 | 维度 |
|------|------|
| **决策模型** | 核实欲 · 决断迟滞 · 掌控补偿 |
| **身体模型** | 身体侵入敏感 · 风险预演度 · 恢复灾难想象 |
| **身份模型** | 自我连续性 · 母职浪漫投射 · 人生节奏保卫度 |
| **关系模型** | 关系确认需求 · 陪伴黏连度 · 承诺验收强度 |
| **现实模型** | 秩序焦虑 · 自由损失敏感 · 照护承重预估 |

### 类型宇宙

- **24 个标准类型**，归属 8 个内部家族：控盘 · 缓刑 · 验收 · 边界 · 献祭 · 投射 · 地基 · 失序
- **第 25 型 `NOIS`（噪点综合体）**：低匹配 + 高矛盾时的幽默兜底型
- 每个类型拥有独立详情页，含全尺寸海报、维度指纹、6 段扩展解读

### 匹配算法

每题 1–5 分，每维 2 题汇总为连续值 → 形成 15 维用户向量 → 与 24 个原型向量计算欧几里得距离（RMSE）→ 取最近原型。支持精确命中计数与三级指纹（低/中/高）可视化。

## 数据与隐私

- **本地优先**：不建账号、不用云端数据库、不接入第三方分析。用户回答保存在当前浏览器 `localStorage`。
- **存储失败保护**：若浏览器禁止存储或空间不足，显示明确错误并阻止继续答题，杜绝静默丢失。
- **版本管理**：数据结构使用版本化键名（`should-we-continue:core-spec-rebuild:v1`），不兼容版本会直接清除旧数据。
- **无 AI 依赖**：Phase 1 不接入任何 LLM，所有结果文案由本地规则 + 模板生成。

## 边界声明

- 本产品是类型测试，不提供医疗、法律或心理建议。
- 不替用户判断"是否继续妊娠"。
- 不对创伤、流产、暴力、自伤或精神疾病内容开玩笑。
- 不收集年龄或生日信息；未成年人妊娠不属于当前可自动筛查的范围，属于已知覆盖缺口。
