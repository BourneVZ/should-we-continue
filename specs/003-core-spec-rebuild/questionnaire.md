# Questionnaire Spec

更新时间：2026-06-28

## 1. 总体结构

- 总题量：30
- 维度数：15
- 每维题量：2
- 题型结构：18 题场景题 + 12 题陈述句
- 作答方式：5 分量表，5 选 1

## 2. 量表定义

所有题目统一使用：

1. 完全不像我
2. 比较不像我
3. 说不好 / 一半一半
4. 比较像我
5. 非常像我

## 3. 维度与题目分配

### 3.1 决策模型

- 核实欲
  - 题 1：发现怀孕后的信息核查冲动
  - 题 2：面对关键不确定性时对证据的需求
- 决断迟滞
  - 题 3：需要拖延决定的倾向
  - 题 4：在压力下更难拍板
- 掌控补偿
  - 题 5：想把变量都排进掌控中
  - 题 6：越慌越想列规则和计划

### 3.2 身体模型

- 身体侵入敏感
  - 题 7：对身体被改变、被占用的敏感
  - 题 8：对身体边界被打破的不适感
- 风险预演度
  - 题 9：会不会提前脑补坏情况
  - 题 10：对身体风险的预演强度
- 恢复灾难想象
  - 题 11：对恢复期的困难想象
  - 题 12：对疼痛、疲惫和失控的放大程度

### 3.3 身份模型

- 自我连续性
  - 题 13：担心“原来的我”断掉
  - 题 14：对身份连续性的保留需求
- 母职浪漫投射
  - 题 15：会不会美化“当妈妈”的画面
  - 题 16：是否容易把母职想象成某种人生意义
- 人生节奏保卫度
  - 题 17：对现有节奏被打乱的抵抗
  - 题 18：是否强烈想守住自己的时间线

### 3.4 关系模型

- 关系确认需求
  - 题 19：是否需要伴侣 / 关系快速给明确态度
  - 题 20：对“被站队”的需求
- 陪伴黏连度
  - 题 21：是否希望有人一直在场
  - 题 22：不被陪着时是否更容易慌
- 承诺验收强度
  - 题 23：是否要求承诺具体落地
  - 题 24：是否更看行动而不是口头支持

### 3.5 现实模型

- 秩序焦虑
  - 题 25：对生活秩序被打乱的焦虑
  - 题 26：对日常流程重排的敏感
- 自由损失敏感
  - 题 27：对自由空间流失的在意程度
  - 题 28：对被固定在某种角色里的抵抗
- 照护承重预估
  - 题 29：对照护劳动重量的预估
  - 题 30：是否先想“这活谁来扛”

## 4. 题目风格

### 4.1 场景题

场景题要求：

- 直接发生在怀孕发现当下或未来育儿想象中
- 有具体动作或画面
- 避免空泛价值判断

### 4.2 陈述句

陈述句要求：

- 用于校准维度
- 避免学术术语
- 避免“我是否抑郁”“我是否焦虑”这类临床暗示

## 5. 禁止内容

- 医疗筛查题
- 心理筛查题
- 家暴 / 自伤 / 自杀 / 创伤直接题
- 真实收支表
- 法律知识题
- 明确去留倾向裁决题
